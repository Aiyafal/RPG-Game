# RPG-Game
类银河恶魔城游戏的具体实现

## 项目说明
本项目为个人Unity学习实践，旨在理解游戏中的完整流程和逻辑

## 已实现功能
-[] 角色移动（InputSystem实现）
-[] 动画状态机（Idle/Move/Run/Dash/Jump/Attack/Death）
-[] 基础战斗系统（普攻连击、受击反馈）
-[] TileMap构建游戏地图
-[] 敌人行为逻辑
-[] 无限背景循环
-[] 异步加载暂停场景（待优化）

## 待开发
-[] 背包系统
-[] 基础UI搭建
-[] 存档系统
-[] Buff系统

## 技术要点
- 使用分层状态机构建角色行为逻辑
- cinemachine构建无限背景
- 采用接口解耦表现和逻辑层

## 运行环境
-Unity 20022.3 LTS
-需要Input System 包
